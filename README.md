# Hoarder
The movie collection tracker web app.

## Built with:
- Next.js
- MovieDB API
- **Database:**
  - MongoDB
  - Mongoose
- **CSS:**
  - Tailwind
- **Localization:**
  - next-intl
- **Authentication:**
  - Next-Auth.js
  - Bcrypt.js
  - Zod

### Notes:
- Auth secret is a Base64-encoded 32-byte random string that can be generated [here](https://generate-secret.vercel.app/32).
